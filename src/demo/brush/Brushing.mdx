## Brush

The `useBrush` hook makes the given element brushable.

<Demo1 />

```tsx
import { useBrush } from 'hyprvis';

function BrushDemo() {
    const ref = useRef();
    const { brush, setBrush } = useBrush(ref);

    return <svg ref={ref}>
        {
            brush ? <BrushRect parent={ref} direction="both" onChange={(brush) => {
            setBrush(brush)
        }} brush={brush} /> : null }
    </svg>
}
```


## Direction

The `direction` prop can control if the brush is only vertical or horizontal, filling the other axis completely.



```tsx
import { useBrush } from 'hyprvis';

function BrushDemo() {
    const ref = useRef();
    const { brush, setBrush } = useBrush(ref);

    return <svg ref={ref}>
        {
            brush ? <BrushRect parent={ref} direction="vertical" onChange={(brush) => {
            setBrush(brush)
        }} brush={brush} /> : null }
    </svg>
}
```
